# 저성취 학생 추출기 (NEIS 일람표용)

나이스(NEIS)에서 내려받은 성적/수행평가 일람표 엑셀 파일을 여러 개 올리면, 평가별 반영 비율을 자동으로 계산해 **합산 총점 기준 저성취 학생**을 뽑아주는 단일 HTML 도구입니다.

## 주소
👉 https://hong-jame.github.io/neis-low-score-filter/

## 특징
- **완전 오프라인 처리**: 업로드한 엑셀 파일은 서버로 전송되지 않고 브라우저 안에서만 처리됩니다. 학생 개인정보가 외부로 나가지 않습니다.
- **형식 자동 인식**
  - 정기시험/수행평가 **교과목별 일람표**(반×번호 매트릭스형)
  - 수행평가 **강의실별 일람표**(학번·성명·합계형)
  - 정기시험 **학생답 정오표**(학번/성명 명부 → 반/번호로 자동 매칭)
- **반영점수 자동 계산 + 수정 가능**: 만점 100점=지필평가, 100점 미만=수행평가로 추정하여 수행평가는 원점수 그대로, 지필평가는 100점 기준으로 환산해 합산합니다. 화면에서 반영점수를 직접 수정할 수 있습니다.
- **집계행/결시/전출 자동 제외**: 응시생수·총점·평균 등 요약행과 자퇴·전출·미인정결 등 텍스트 값은 자동으로 걸러냅니다.
- **결과 엑셀 다운로드** 지원

## 사용법
1. 위 주소로 접속(또는 `index.html`을 브라우저에서 열기)
2. 성적 파일들(+선택적으로 학번/성명 명부)을 끌어다 놓기
3. 자동 계산된 반영점수 확인 후, 필요하면 수정
4. 기준 점수(기본 40점) 이하 학생 명단 확인 및 엑셀 다운로드

## 기술
- 단일 HTML 파일 (외부 의존성 없음, SheetJS 라이브러리 내장)
